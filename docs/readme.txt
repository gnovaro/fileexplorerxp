SETUP INSTRUCTIONS
==================
1 - Descompress file in your web server
2 - Open Web browser in: http://YOURSITE/fileexplorerxp/
    This operations run setup.php and create config.php
    OR
    Edit config_sample.php
    * Set URL for your website
    * User and Password (DEFAULT user:admin pass:admin)
    And save as "config.php"

Requisites
==========
* Apache or other web server
* Php 5 or later
